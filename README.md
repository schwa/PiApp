# PiApp

A toy macOS/iOS app to experiment with [Miguel de Icaza](https://github.com/migueldeicaza)'s [PiSwift](https://github.com/xibbon/PiSwift) - a Swift implementation of an AI coding agent.

## Features

- **Chat**: Interact with Claude via the Anthropic API (supports both API keys and OAuth)
- **Files**: Browse and manage files
- **Terminal**: Embedded terminal emulator (macOS only, using [SwiftTerm](https://github.com/migueldeicaza/SwiftTerm))

## Requirements

- macOS 15+ / iOS 18+
- Xcode 26+
- Anthropic API key or Claude Code subscription

## Setup

1. Clone the repo
2. Open `PiApp.xcodeproj`
3. Build and run
4. Configure your API key in Settings (Cmd+, on macOS, Settings tab on iOS)

## Status

This is a playground project, not production software.
